# Hero Dashboard Reel Assets

Drop these files into `public/` to activate the homepage hero video:

- `hero-dashboard-reel-desktop.mp4`
- `hero-dashboard-reel-desktop.webm`
- `hero-dashboard-reel-mobile.mp4`
- `hero-dashboard-reel-poster.svg` or a higher-fidelity poster replacement

Recording flow:

1. Chat
2. Dashboard
3. Inbox
4. Map
5. Calendar day view

Recording rules:

- Use fake/demo data only.
- Fill every captured screen with believable jobs, contacts, and metrics.
- Record at a stable browser size that matches a 16:9 hero crop.
- Keep cursor movement deliberate and minimal.
- Avoid empty states, beta labels, debug UI, or stale branding.

Edit targets:

- Desktop loop: 12-18 seconds
- Mobile loop: shorter and lighter
- Autoplay-safe: muted, no audio dependence
- Start strong in the first 2-3 seconds
